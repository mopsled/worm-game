Instructions

Controls

  • Arrow Keys - Player 1
  • W, A, S, D - Player 2

Unusual controls

  • p - Pause the game
  • 1 - Set the game to one-player mode and reset
  • 2 - Set the game to two-player mode and reset

Testing controls

  • g - Grow all snakes on the board
  • \ - Reset the game and high scores
