Collaborative two-player
item-driven snake

We've taken the classic game of snake and updated it, adding a large number of features to make a challenging, innovative game. We've thrown aside imperative programming and created a beautifully simple system of functionally-defined items. We've implemented a variety of these items, including items that grow, shrink, destroy, slow, and even teleport a player's snake across the board. Even better, we've made new items incredibly easy to create. Just check out the code needed to create a portal object, taken from our project:

57    var PORTAL_ACTION = function(player) {
58    	worms[player].position.x = getRandomX();
59    	worms[player].position.y = getRandomY();
60    }

Also, we've changed the classical snake paradigm to a collaborative two-player game. Players must work together to coordinate collecting food, sharing items, and avoiding collision as they struggle for a high score.
