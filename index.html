Collaborative two-player item-driven snake

We've taken the classic game of snake and updated it, adding a large number of features to make a challenging, innovative game. We've thrown aside imperative programming and created a beautifully simple system of functionally-defined items. We've implemented a variety of these items, including items that grow, shrink, destroy, slow, and even teleport each player's snake across the board. Even better, we've made new items incredibly easy to create. Just check out the code needed to create our portal object:

var PORTAL_ACTION = function(player) {
    worms[player].position.x = getRandomX();
    worms[player].position.y = getRandomY();
};

Also, we've changed the classical snake paradigm to a collaborative two-player game. Players must work together to coordinate collecting food, sharing items, and avoiding collision as they struggle for a high score.
